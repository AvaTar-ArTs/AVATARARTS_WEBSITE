# AvatarArts Website Images

This directory contains images for the AvatarArts website at avatararts.org.

## Image Descriptions

### hero-placeholder.jpg
A placeholder image representing the creative automation platform. This would typically be replaced with a high-quality image representing music, AI, or creative technology.

### favicon.ico
The website's favicon - a small icon that appears in browser tabs and bookmarks.

## Image Requirements

For production use, the following image specifications should be followed:

- **Hero Images**: 1920x1080 pixels, JPEG format, optimized for web
- **Icons**: SVG format preferred, PNG as fallback
- **Favicon**: 16x16, 32x32, and 48x48 pixel sizes in ICO format
- **Social Media**: 1200x630 pixels for Open Graph, 400x400 for Twitter

## Placeholder Images

For development purposes, placeholder images can be used from services like:
- Unsplash (unsplash.com)
- Pexels (pexels.com)
- Placehold.co (placehold.co)

## Image Optimization

All images should be optimized for web delivery:
- Compress images without significant quality loss
- Use appropriate file formats (JPEG for photos, PNG for graphics with transparency, SVG for icons)
- Implement lazy loading for images below the fold
- Use responsive images with srcset attributes
